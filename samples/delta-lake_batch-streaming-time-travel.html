Migrating to Delta Lake¶

To convert your table to a Delta Lake table, the easiest method is to:

  • Load your data to create a DataFrame
  • Save it using the df.write.format("delta").mode(...).save(...) method
  • Load it using spark.read.format("delta").load(...)

Create insert_rows DataFrame for our New Flights¶

The insert_rows DataFrame contains the new flights that we want to batch add to our Delta Lake table. We will add 10 batches of data into our Delta Lake table departureDelays.

In [11]:
# Batch Insert flights
i = 1
while i <= 10: 
    # Generate new delays
    delay_a = 500 + 20*i
    delay_b = 500 + 30*i
    
    # Create new flights
    newflights =  [(4011028, delay_a, 590, 'SEA', 'SFO'), (4021028, delay_b, 590, 'SEA', 'SFO')] 
    cols = ['date', 'delay', 'distance', 'origin', 'destination']

    # Create `insert_rows` DataFrame
    insert_rows = spark.createDataFrame(newflights, cols)

    # Convert LongType() to IntegerType()
    insert_rows = insert_rows \
        .withColumn("date", col("date").cast(IntegerType())) \
        .withColumn("delay", col("delay").cast(IntegerType())) \
        .withColumn("distance", col("distance").cast(IntegerType()))

    # Display data
    insert_rows.toPandas()
    
    # Batch write
    insert_rows.write.format("delta").mode("append").save("departureDelays.delta") 
    
    # Loop through
    i += 1

Let's Travel back in Time!¶

Databricks Delta’s time travel capabilities simplify building data pipelines for the following use cases.

  • Audit Data Changes
  • Reproduce experiments & reports
  • Rollbacks

As you write into a Delta table or directory, every operation is automatically versioned.

You can query by:

  1. Using a timestamp
  2. Using a version number

using Python, Scala, and/or SQL syntax; for these examples we will use the Python syntax.

For more information, refer to Introducing Delta Time Travel for Large Scale Data Lakes

Review Delta Lake Versions¶

The following code shows the 11 different views of the data associated with the different versions of data using the option:

.option("versionAsOf", version)

This will initially load the delays list so we can plot the numpy arrays directly.

In [78]:
# create empty list
delays = []

# Indicating the first version 
version = 0 
vmax = 10

while (version <= vmax):
    # Read data
    delays_delta_tt = spark.read.format("delta").option("versionAsOf", version).load("departureDelays.delta")

    # Review data
    sea2sfo = delays_delta_tt \
        .where(col("origin") == 'SEA') \
        .where(col("destination") == 'SFO') \
        .where(col("delay") > 180)

    # Convert to Pandas for plotting purposes
    sea2sfo_pdf = sea2sfo.toPandas()

    # Extract delay values to plot histogram
    delays.append(sea2sfo_pdf.loc[:, ['delay']].values)
    
    # Looping
    version += 1
